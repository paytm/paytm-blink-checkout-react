# Build creation

The build could be created via following npm scripts : 

### build-umd
 The command creates UMD build.

Build file location :

> lib/build/umd

### build-common
 The command creates common js build.

Build file location :

> lib/build/commonjs

### build-all
 The command creates both UMD and common js build for  paytm-blink-checkout-react.

Build file location :

> lib/build/
